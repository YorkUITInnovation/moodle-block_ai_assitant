Instructions for Creating a Word Question Template

  1. Add Questions as Headings
    • Each question must be set as a heading.
    • Example: Heading1
  2. Provide Alternative Phrasings
    • Under each heading, list examples of other ways to ask the question.
    • Example:
      • How do I create a Word Question template?
      • Can you help me with a Word Question template?
  3. Response Section
    • Do not delete: The response to any of these questions or prompts is:
    • Beneath that sentence, add your answer.
    • Example:
      • The response to any of these questions or prompts is:
      • You can create a Word Question template by following these steps...
  4. Repeat for Each Question
    • Repeat the same steps for every question you want to include.
  5. Upload Document
    • Once the document is ready, click on Custom Questions to upload your document.
    • Allow time for the AI Assistant to train itself on the questions.
